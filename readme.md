# Annotated emotion process 
> Author: LiuZiyu

> Created date: 2024/11/15

## Introduction
This is a project to process a_emotion (Valence and Arousal) in 2024.
## Files Content
This repo includes boris projects, annotated V and A data, calculated kappa and scatter plot, and R scripts.

## Workflow
1. Boris 项目文件（project）上传到 documents 文件夹下的 projects 文件夹，命名方式参考已有文件；
2. Boris 导出的 excel 文件上传到 0_rawdata 文件夹下.
